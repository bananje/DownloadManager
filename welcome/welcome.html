Shape your downloads with smart filters

The extension lets you manage files through a set of advanced filters — sort, narrow down by date and pick just the file types you need.

Filters you can combine

  • Sort by name Arrange files alphabetically, ascending or descending.
  • Sort by size Find the heaviest or the lightest files at a glance.
  • Date range Pick today, this week, this month or a custom interval.
  • File types Show only images, video, archives, programs or documents.

Active filters appear as chips at the top of the popup — remove them at any time.
