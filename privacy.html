Privacy Policy

This page explains what information the Download Manager Chrome extension uses, how it is processed, and what is never collected, sold, or shared.

Last updated: April 27, 2026

In short

  • The extension does not collect, store, or transmit your personal information.
  • It does not use analytics, telemetry, tracking pixels, or third‑party services.
  • All data (settings, download history filters) stays locally on your device.
  • There is no remote server; the extension does not "phone home".
  • Your downloads, browsing activity, and files are never sold or shared.

1. Who we are

"Download Manager" (the "Extension") is a browser extension that helps you organize transfers, search files, filter results, and keep your download history clean and easy to manage. This document is the privacy policy of the Extension's developer ("we", "us").

2. What data the Extension accesses

To work, the Extension uses Chrome APIs that, by their nature, expose certain information locally inside your browser. None of this information leaves your device.

downloads Read your local download history (file name, URL, status, size, progress) to display and manage it inside the popup.
downloads.open Open a finished file when you click it in the list.
downloads.ui Show the file in the system file manager ("Show in Explorer").
tabs Detect the current tab so the in‑page download notification can be displayed in the right place.
scripting Inject the lightweight in‑page toast that confirms a download has started.
storage Save your settings (such as preferred filters and UI options) locally via chrome.storage.local.
alarms Schedule lightweight background tasks (e.g. periodic UI refresh of in‑progress downloads).
host_permissions: <all_urls> Required so the in‑page download notification can appear on any site where you initiate a download. The Extension does not read page content.

3. What we do not do

  • We do not collect personally identifiable information (name, email, phone, address).
  • We do not collect health, financial, authentication, personal communications, or location data.
  • We do not read, store, or transmit the content of web pages you visit.
  • We do not track your browsing activity or build any profile of you.
  • We do not use cookies, fingerprinting, or third‑party trackers.
  • We do not sell, rent, or share data with third parties — there is no data to share.
  • We do not use your data for advertising or for purposes unrelated to the Extension's single core feature (download management).
  • We do not use your data to determine creditworthiness or for lending.

4. Local storage

The Extension stores small pieces of information using Chrome's built‑in chrome.storage.local API. This data lives only in your browser profile on your computer. Examples:

  • UI preferences (selected filter, sort order, theme);
  • Cached metadata about downloads already known to Chrome (so the list can render quickly).

You can erase this data at any time by removing the Extension or by clearing the extension's storage in Chrome.

5. Network usage

The Extension itself does not contact any server. The only network activity that may be triggered through it is the actual file download, which is performed by Chrome to the URL you chose. We have no servers, no backend, and no analytics endpoints.

6. Permissions justification (Chrome Web Store)

Each requested permission is used exclusively for the single purpose of managing downloads inside the user's browser. No permission is used to gather data for any other purpose.

7. Children's privacy

The Extension is a general‑purpose tool and is not directed at children. Because no personal data is collected, no information about children is collected either.

8. Changes to this policy

If this policy changes, the updated version will be published at the same URL with a new "Last updated" date. Significant changes may also be reflected in the Extension's listing on the Chrome Web Store.

9. Contact

If you have any questions about this Privacy Policy, please open an issue in the project's GitHub repository or contact the developer through the support channel listed on the Chrome Web Store page.

Политика конфиденциальности

Эта страница объясняет, какую информацию использует расширение Download Manager, как она обрабатывается и что мы никогда не собираем, не продаём и не передаём третьим лицам.

Последнее обновление: 27 апреля 2026 г.

Кратко

  • Расширение не собирает, не хранит и не передаёт ваши персональные данные.
  • Не используется аналитика, телеметрия, пиксели и сторонние сервисы.
  • Все данные (настройки, фильтры) хранятся локально на вашем устройстве.
  • Нет удалённого сервера — расширение никуда ничего не отправляет.
  • Информация о загрузках, посещениях и файлах не продаётся и не передаётся.

1. Кто мы

«Download Manager» (далее — «Расширение») — это расширение для браузера, которое помогает упорядочивать загрузки, искать файлы, фильтровать результаты и поддерживать историю загрузок в чистом и удобном виде. Этот документ — политика конфиденциальности разработчика Расширения («мы»).

2. К каким данным обращается Расширение

Для работы Расширение использует API Chrome, которые по своей природе предоставляют доступ к некоторой информации локально внутри браузера. Эта информация не покидает ваше устройство.

downloads Чтение локальной истории загрузок (имя файла, URL, статус, размер, прогресс) для отображения и управления списком в окне расширения.
downloads.open Открытие готового файла по клику в списке.
downloads.ui Показ файла в системном файловом менеджере («Показать в Проводнике»).
tabs Определение активной вкладки, чтобы уведомление о начале загрузки появилось в нужном месте.
scripting Внедрение лёгкого всплывающего уведомления на странице о начавшейся загрузке.
storage Сохранение ваших настроек (фильтры, параметры интерфейса) локально через chrome.storage.local.
alarms Запуск лёгких фоновых задач (например, периодическое обновление активных загрузок).
host_permissions: <all_urls> Нужно для отображения уведомления о загрузке на любом сайте, где вы её начали. Содержимое страниц не читается.

3. Что мы не делаем

  • Не собираем персональные данные (имя, e‑mail, телефон, адрес).
  • Не собираем медицинские, финансовые, аутентификационные данные, личную переписку и геолокацию.
  • Не читаем, не храним и не передаём содержимое посещаемых вами веб‑страниц.
  • Не отслеживаем вашу активность и не строим никаких профилей пользователя.
  • Не используем cookies, fingerprinting и сторонние трекеры.
  • Не продаём, не сдаём в аренду и не передаём данные третьим лицам — передавать просто нечего.
  • Не используем ваши данные для рекламы или целей, не связанных с основной функцией Расширения (управление загрузками).
  • Не используем ваши данные для оценки кредитоспособности или кредитования.

4. Локальное хранилище

Расширение сохраняет небольшие данные через встроенный API Chrome chrome.storage.local. Эти данные остаются только в вашем профиле браузера на вашем компьютере. Примеры:

  • настройки интерфейса (выбранный фильтр, сортировка, тема);
  • кэш метаданных о загрузках, уже известных Chrome (чтобы список открывался быстро).

Эти данные можно удалить в любой момент: удалите Расширение или очистите его хранилище в настройках Chrome.

5. Сетевая активность

Само Расширение не обращается ни к каким серверам. Единственная сетевая активность, которая может произойти через него, — это фактическая загрузка файла, инициированная Chrome по URL, который выбрали вы. У нас нет ни серверов, ни бэкенда, ни эндпоинтов аналитики.

6. Обоснование разрешений (Chrome Web Store)

Каждое запрошенное разрешение используется исключительно для одной цели — управления загрузками внутри браузера пользователя. Ни одно разрешение не используется для сбора данных в других целях.

7. Конфиденциальность детей

Расширение — это инструмент общего назначения и не предназначено специально для детей. Так как никакие персональные данные не собираются, информация о детях также не собирается.

8. Изменения политики

Если эта политика изменится, обновлённая версия будет опубликована по тому же адресу с новой датой «Последнее обновление». О существенных изменениях может сообщаться также на странице Расширения в Chrome Web Store.

9. Контакты

Если у вас есть вопросы по этой политике конфиденциальности, создайте issue в GitHub‑репозитории проекта или напишите разработчику через канал поддержки, указанный на странице расширения в Chrome Web Store.

Datenschutzerklärung

Diese Seite erklärt, welche Informationen die Chrome-Erweiterung Download Manager verwendet, wie sie verarbeitet werden und was niemals erfasst, verkauft oder weitergegeben wird.

Letzte Aktualisierung: 27. April 2026

Kurz gesagt

  • Die Erweiterung erfasst, speichert oder übermittelt keine personenbezogenen Daten.
  • Sie verwendet keine Analysen, Telemetrie, Tracking-Pixel oder Drittanbieterdienste.
  • Alle Daten (Einstellungen, Filter des Download-Verlaufs) verbleiben lokal auf Ihrem Gerät.
  • Es gibt keinen Remote-Server; die Erweiterung „telefoniert nicht nach Hause".
  • Ihre Downloads, Browseraktivitäten und Dateien werden niemals verkauft oder weitergegeben.

1. Wer wir sind

„Download Manager" (die „Erweiterung") ist eine Browser-Erweiterung, die Ihnen hilft, Übertragungen zu organisieren, Dateien zu suchen, Ergebnisse zu filtern und Ihren Download-Verlauf sauber und übersichtlich zu halten. Dieses Dokument ist die Datenschutzerklärung des Entwicklers der Erweiterung („wir", „uns").

2. Worauf die Erweiterung zugreift

Damit die Erweiterung funktioniert, verwendet sie Chrome-APIs, die naturgemäß bestimmte Informationen lokal in Ihrem Browser bereitstellen. Keine dieser Informationen verlässt Ihr Gerät.

downloads Lesen Ihres lokalen Download-Verlaufs (Dateiname, URL, Status, Größe, Fortschritt) zur Anzeige und Verwaltung im Popup.
downloads.open Öffnen einer fertigen Datei beim Klick in der Liste.
downloads.ui Datei im System-Dateimanager anzeigen („Im Explorer anzeigen").
tabs Erkennen des aktuellen Tabs, damit die Download-Benachrichtigung an der richtigen Stelle angezeigt werden kann.
scripting Einfügen des leichten In-Page-Toasts, der den Beginn eines Downloads bestätigt.
storage Speichern Ihrer Einstellungen (z. B. Filter und Oberflächenoptionen) lokal über chrome.storage.local.
alarms Planen leichter Hintergrundaufgaben (z. B. periodische Aktualisierung laufender Downloads).
host_permissions: <all_urls> Erforderlich, damit die Download-Benachrichtigung auf jeder Website angezeigt werden kann, auf der Sie einen Download starten. Die Erweiterung liest keine Seiteninhalte.

3. Was wir nicht tun

  • Wir erfassen keine personenbezogenen Daten (Name, E-Mail, Telefon, Adresse).
  • Wir erfassen keine Gesundheits-, Finanz-, Authentifizierungs-, persönlichen Kommunikations- oder Standortdaten.
  • Wir lesen, speichern oder übermitteln keine Inhalte der von Ihnen besuchten Webseiten.
  • Wir verfolgen Ihre Browsing-Aktivitäten nicht und erstellen kein Profil von Ihnen.
  • Wir verwenden keine Cookies, kein Fingerprinting und keine Drittanbieter-Tracker.
  • Wir verkaufen, vermieten oder teilen keine Daten mit Dritten – es gibt nichts zu teilen.
  • Wir verwenden Ihre Daten nicht für Werbung oder Zwecke, die nicht mit der Kernfunktion der Erweiterung (Download-Verwaltung) zusammenhängen.
  • Wir verwenden Ihre Daten nicht zur Bonitätsbeurteilung oder Kreditvergabe.

4. Lokaler Speicher

Die Erweiterung speichert kleine Informationen über die in Chrome integrierte API chrome.storage.local. Diese Daten verbleiben nur in Ihrem Browser-Profil auf Ihrem Computer. Beispiele:

  • Oberflächeneinstellungen (gewählter Filter, Sortierreihenfolge, Theme);
  • Zwischengespeicherte Metadaten zu bereits in Chrome bekannten Downloads (damit die Liste schnell gerendert wird).

Sie können diese Daten jederzeit löschen, indem Sie die Erweiterung entfernen oder den Speicher der Erweiterung in Chrome löschen.

5. Netzwerknutzung

Die Erweiterung selbst kontaktiert keinen Server. Die einzige Netzwerkaktivität, die durch sie ausgelöst werden kann, ist der eigentliche Datei-Download, der von Chrome an die von Ihnen gewählte URL durchgeführt wird. Wir haben keine Server, kein Backend und keine Analyse-Endpunkte.

6. Begründung der Berechtigungen (Chrome Web Store)

Jede angeforderte Berechtigung wird ausschließlich für den einen Zweck verwendet, Downloads im Browser des Nutzers zu verwalten. Keine Berechtigung wird für andere Zwecke verwendet.

7. Datenschutz von Kindern

Die Erweiterung ist ein Allzweck-Werkzeug und richtet sich nicht an Kinder. Da keine personenbezogenen Daten erfasst werden, werden auch keine Informationen über Kinder erfasst.

8. Änderungen dieser Richtlinie

Bei Änderungen dieser Richtlinie wird die aktualisierte Version unter derselben URL mit einem neuen „Letzte Aktualisierung"-Datum veröffentlicht. Wesentliche Änderungen können auch im Eintrag der Erweiterung im Chrome Web Store widergespiegelt werden.

9. Kontakt

Wenn Sie Fragen zu dieser Datenschutzerklärung haben, öffnen Sie bitte ein Issue im GitHub-Repository des Projekts oder kontaktieren Sie den Entwickler über den auf der Chrome Web Store-Seite angegebenen Support-Kanal.

隐私政策

本页说明 Chrome 扩展程序 Download Manager 使用哪些信息、如何处理这些信息，以及哪些内容绝不会被收集、出售或共享。

最后更新：2026 年 4 月 27 日

简而言之

  • 本扩展程序不会收集、存储或传输您的个人信息。
  • 不使用任何分析、遥测、追踪像素或第三方服务。
  • 所有数据（设置、下载历史筛选条件）都保存在您的设备本地。
  • 没有远程服务器；本扩展程序不会"回拨主控"。
  • 您的下载、浏览活动和文件绝不会被出售或共享。

1. 我们是谁

"Download Manager"（"本扩展程序"）是一款浏览器扩展程序，可帮助您整理传输、搜索文件、筛选结果，并保持下载历史记录干净且易于管理。本文档是本扩展程序开发者（"我们"）的隐私政策。

2. 本扩展程序访问哪些数据

为了正常工作，本扩展程序使用 Chrome API，这些 API 本身会在您的浏览器内本地公开某些信息。这些信息不会离开您的设备。

downloads 读取您本地的下载历史（文件名、URL、状态、大小、进度），以便在弹出窗口中显示和管理。
downloads.open 在您单击列表中已完成的文件时打开它。
downloads.ui 在系统文件管理器中显示文件（"在资源管理器中显示"）。
tabs 检测当前标签页，以便页面内的下载通知能够显示在正确的位置。
scripting 注入轻量级页面内提示，确认下载已开始。
storage 通过 chrome.storage.local 在本地保存您的设置（例如首选筛选条件和界面选项）。
alarms 安排轻量级后台任务（例如定期刷新进行中的下载界面）。
host_permissions: <all_urls> 必需的，以便在您启动下载的任何网站上显示页面内下载通知。本扩展程序不会读取页面内容。

3. 我们不做的事情

  • 我们不收集任何个人身份信息（姓名、电子邮件、电话、地址）。
  • 我们不收集健康、财务、身份验证、个人通信或位置数据。
  • 我们不读取、存储或传输您访问的网页内容。
  • 我们不追踪您的浏览活动，也不为您建立任何档案。
  • 我们不使用 Cookie、指纹识别或第三方追踪器。
  • 我们不向第三方出售、出租或共享数据——根本没有数据可共享。
  • 我们不将您的数据用于广告或与本扩展程序唯一核心功能（下载管理）无关的任何目的。
  • 我们不将您的数据用于评估信用或贷款。

4. 本地存储

本扩展程序使用 Chrome 内置的 chrome.storage.local API 存储少量信息。这些数据仅存在于您计算机上的浏览器配置文件中。示例：

  • 界面偏好（所选筛选条件、排序方式、主题）；
  • 已知下载的缓存元数据（以便列表能够快速渲染）。

您可以随时删除这些数据：删除本扩展程序，或在 Chrome 中清除扩展程序的存储。

5. 网络使用

本扩展程序本身不会联系任何服务器。通过它可能触发的唯一网络活动是实际的文件下载，该下载由 Chrome 执行到您选择的 URL。我们没有服务器、没有后端，也没有任何分析端点。

6. 权限说明（Chrome 网上应用店）

所请求的每项权限都仅用于在用户浏览器内管理下载这一单一目的。任何权限都不会被用于为其他目的收集数据。

7. 儿童隐私

本扩展程序是通用工具，并非专门面向儿童。由于不收集任何个人数据，因此也不会收集儿童的任何信息。

8. 政策变更

如果本政策发生变更，将在同一 URL 发布更新版本，并附有新的"最后更新"日期。重大变更也可能反映在 Chrome 网上应用店的扩展程序详情页中。

9. 联系方式

如果您对本隐私政策有任何疑问，请在项目的 GitHub 存储库中开启 issue，或通过 Chrome 网上应用店页面上列出的支持渠道联系开发者。
